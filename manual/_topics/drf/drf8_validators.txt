8. Validators (Валидаторы)




1. UniqueValidator(
    queryset, # Набор запросов для валидации.
    message, # Сообщение об ошибке при неудачной проверке.
    lookup, # Нахождение существующего экземпляра с проверямым значением. 
            # По умолчанию 'exact'.
)
    Валидатор используется для ограничения unique=True.
2. UniqueTogetherValidator(
    queryset,
    fields, # Список имен полей, которые должны быть уникальны.
    message,
)
    Валидатор используется для наложения ограничений unique_together.
3. UniqueForDateValidator()
4. UniqueForMonthValidator()
5. UniqueForYearValidator(
    queryset,
    field, # Имя поля для валидации в диапазоне дат.
    date_field, # Имя поля для валидации в диапазоне дат.
    message,
)
6. CurrentUserDefault()
    Может быть использован для представления текущего пользователя. Чтобы 
    использовать его, request должен быть предоставлен как часть контекстного 
    словаря при инстанцировании сериализатора.
7. CreateOnlyDefault()
    Может быть использован только для установки аргумента по умолчанию во время 
    операций создания. При обновлении поле опускается.
